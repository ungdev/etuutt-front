# Installation

## Requirement :

- React Native with expo

## Installation :

### clone repository

- in a terminal :
- git clone https://github.com/ungdev/etuutt-front
- cd etuutt-front

### install dependencies :

- yarn

### launch the app on port 8080 :

- yarn dev
